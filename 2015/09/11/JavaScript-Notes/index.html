JavaScript 入门学习笔记

發表於 2015-09-11   |   分類於 Notes   |  

　　HTML 就像是人的身体； CSS 就像是各种各样的衣服，能给人带来各种样子上的变化； Javascript 就像是人的活动，可以改变身体的构成，也可以改变各种衣服的样子，三者结合就给网页带来了无穷的变化，成就无限可能。

　　上面是摘自 百度知道的一个问答 ，通俗的把 HTML , CSS , JavaScript 的关系描述了一遍，贴切而且挺有意思的。下面使用栗子讲解的方式把我的 JavaScript 学习过程记录下来，以后忘了在回头看看 : )

　　首先说明两点：

  • JavaScript 是用来在页面中特效的，基于对象，并且和 HTML/CSS 一样都是由浏览器解析的。
  • JQuery 是 一个 JavaScript 库，使用它简化了 JS 的编程，并且解决了浏览器之间的兼容问题。

0x 00 嵌入格式

<script language = "javascript" type = "text/javascript">
    // JS 代码
</script>

　　在这里，属性 language 和 type 可以不写（默认 JavaScript ），但要是嵌入其他前段脚本，比如 jscript，需要写上 language，type 。

　　嵌入格式可以嵌入 , 标签内部，或者 外，不能够嵌入到 CSS 里。

0x 01 输出

document.write("打印一个字符串!")  // 在文档中输出双引号中的字符串

alert(/hello \n form:www.evilclay.com/)  // 弹出有换行符(\n)的 警告框

　　在这里除了 alert() 是用来弹出提示框，还有下面这两个函数。

var r = confirm("Are You sure?")  // 确认框，返回 OK 或者 Cancel
var name = prompt("Please input You name,baby:","DaGe") // 有默认值的  提示框，点击确定返回输入的值，点击取消 返回 null

0x 02 如何使用

　　主要分为以下 4 种情况：

  • 使用<script> </script>标签将 JS 嵌入到 HTML 中。
  • 在 href 属性中 引入， 如: href = "javascript:alert(/evilclay.com/)"
  • 在事件中引入，如 onclick 事件。
  • 在文档中的任意位置调用外部 JS 文件，如：<script src = "test.js"> </script>

0x 03 变量

　　声明变量要注意的 3 点：

  • 变量绝大多数情况下以字母开头。
  • 变量也可以以 $ 和 _（下划线） 开头，但不推荐这么做。
  • 现阶段可以把带有 var 声明理解为局部变量，不带的是全局变量。

　　数组的两种声明方式：

1. var cars = new Array();

　　cars[0] = "Audi" ; cars[1] = "BMW";

2. var cars = new Array("Audi","BMW");
　　// 和上面的声明效果是一样的。

0x 04 对象

var person = {
　　　　firstname:"Bill",
　　　　lastname:"Gates"
};

　　上面的栗子初始化了一个对象 person ，对象属性有下面两种寻址方式:

- name1 = person.firstname;
- name2 = person['lastname']

　　JS 中有以下几种 变量类型：

1. var str = " hello ";  // 字符串
2. var num = 23333 ;  // 数字
3. var result = true;  // 布尔
4. var arr = new Array();  // 数组
5. var ob = { name:"evilclay", age:22 };  // 对象

　　在 JS 中，所有的变量 都可以看做是对象（ 属性 + 方法 ）。

　　对于字符串(String) 变量，有一些内建的属性和方法：

var txt = "hell0"
txt.Length  // 5 , 获得长度
txt.indexOf("el")  // 1 , 用来定位某个字符串在原字符串第一次出现的位置
txt.replace("0","0") // hello, 字符串替换
txt.search("ll")  // 2, 返回字符串第一次匹配的位置，和 indexOf 有点区别:)
txt.toUpperCase() // HELL0, 转换成大写

　　遍历对象的属性（ for ..in 结构 ）:

var person = { firstname:"Bill",lastname:"Gates" }
var x; var txt = "";
for (x in person){
    txt = txt + person[x];
}  
// 输出： BillGates

0x 05 函数

　　只有在被调用时，函数体中的代码才会执行。
　　函数的定义：

function myFunction(str){
    alert(str);
    return str; // 返回值可有可无，看情况
}

0x 06 运算符

  • 算数运算符 + * / % ++ –
  • 赋值运算符 = += -= *= /= %=
  • 条件运算符 > < == != >= <= === !==
  • 逻辑运算符 && || ！
  • 位运算符 >> >>> << <<< & | ~
  • 其他运算符 ?:

　　说明：

　　逻辑运算符有短路原则。

　　== 是 非严格匹配，也就是说匹配前会转换类型，比如 true == 1 就是正确的。

　　=== 是严格匹配，匹配前会先判断类型是否一致。

　　>>> 是无符号右移，比如 100 >>> 2 结果是 25，需要把100转换成二进制右移2位。

　　~ 是取反操作。

　　有趣的小栗子：

a = 10 + 2 + "3abc"  // 在 JS 中 输出 "1023abc"
a = 10 + 2 + "3abc"  // 在 PHP 中 输出 15

0x 07 DOM

　　DOM —— Document Object Module ，文档对象模型

　　JS 可以修改 HTML 元素， HTML 属性， CSS 样式。

查找 HTML 元素

  1. via id / class

var x = document.getElementById("Intro");

  1. via 标签名

var x = document.getElementById("stu");
var y = x.getElementByTagName("p") // 查找 所有<p>中 id = "stu" 的标签

改变 HTML 输出流

　　使用 document.write() 方法 // 重新加载后会覆盖原文档

改变 HTML 属性

　　语法： document.getElementById(id).attribute = new value;

eg: document.getElementById("pic").href = "http://www.evilclay.com/a.jpg";

改变 HTML 样式

　　语法： document.getElementById(id).style.property = new style;

eg: document.getElementById("p2").style.color = "blue";

使用 JS 来分配事件

　　语法： document.getElementById(id).event = function();

eg: 　document.getElementById(id).event = function();

　　JS 中 常用的事件 如下：

  • onload & onunload

　　进入、离开页面时触发， onload 事件可用于检测访问者的浏览器类型，版本

　　onload 与 onunload 可用于处理cookie

  • onchange

　　在文本框输入完信息焦点离开后会触发事件

　　可用来做输入检查

  • onmouseover & onmouseout

　　鼠标移至、移出 HTML 元素时触发事件

  • onmousedown & onmouseup

　　鼠标点击、松开 HTML 元素时触发事件

  • onfocus

　　当输入字段获得焦点时触发事件

　　常用于表单提示

　　注： 使用 JS 还可以增加、删除 HTML 元素。

0x 07 常用的内建对象

Date() 对象

　　用来返回当日的时间

var d1 = new Date()  # 定义 d1 为当前时刻

var d2 = new Date("2015/9/20 00:33:00")  # 使用字符串形式定义时间

var d3 = new Date(2015,9,20,0,33,0)  # 使用数字形式定义时间（至少 3 个数字）

var v1 = d1.toLocaleString()  # 获得 d1 的 本地时间表示

var v2 = d1.getTime()  # 获得 d1 的毫秒表示形式(从 1970.0.0 00:00:00 开始计算)

random()

　　用来返回 0-1 之间的随机数

Math() 对象

　　用来进行数学计算

　　获得 n1 - n2 之间的随机整数：

var s = Math.floor(Math.random()*(n2 - n1 + 1)) + n1;

String() 对象

var str = "abcd"; or var str = new String("abcd")

var v1 = str.charAt(3)  # 取位置为3的字符（从0开始）

var v3 = str.subString(n,m)  # 取 n 到 m 之间的字符(含n不含m)

var v4 = str.replace("cd","dc")  # 替换字符串

var v5 = str.toUpperCase()  # 大写

var v6 = str.toLowerCase()  # 小写

var v7 = str.split("b")  # "b"作为分隔符分割字符串得到数组

var p1 = str.indexOf("c")  # 获得 字符串 c 首次出现的位置

var p2 = str.lastIndexof("c")  # 获得字符串 c 最后一次出现的位置

数组

var arr = [[1,2,3],[4,5],[6,7]]

arr[0].length  # 获得数组 arr[0] 的长度 ，为 3

　　常用方法：

concat(),join(),pop(),push(),shift(),unshift(),reverse()

event 对象

event.clientX  # 获得网页上鼠标的 x 坐标

event.clientY  # 获得网页上鼠标的 y 坐标

event.keyCode  # 键盘事件的按键值

event.target  # Forefox 中，事件发生的那个对象(事件源)

event.srcElement  # IE中的事件源
