Shadowsocks服务端配置命令

By Anka9080

发表于 2017-03-11

文章目录

  1. 安装 服务端
    pip install shadowsocks

  2. 创建 /etc/shadowsocks/，进入并添加 config.json文件，内容如下：
    {
    服务器IP地址需要设置为0.0.0.0
    “server”:”0.0.0.0”,
    “server_port”:8838,
    “local_port”:1080,
    “password”:”pass”,
    “timeout”:600,
    “method”:”aes-256-cfb”
    }

  3. 安装aes加密相关库
    apt-get install python-m2crypto

  4. 设置开机启动 打开 /etc/rc.local，并添加如下内容：
    /usr/local/bin/ssserver -c /etc/shadowsocks/config.json
